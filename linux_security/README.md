# Linux Security Basics

This project contains bash scripts for Linux security basics, including firewall configuration and connection monitoring.

## Scripts

- **0-login.sh** - Login security
- **1-active-connections.sh** - Monitor active connections
- **2-incoming_connections.sh** - Firewall rules for incoming TCP port 80

## Requirements

- Kali Linux
- Root or sudo privileges
- Bash shell

## Usage

Run scripts with sudo:

```bash
sudo ./script_name.sh
```

## Notes

All scripts follow Betty style guidelines and must be exactly 2 lines long.
